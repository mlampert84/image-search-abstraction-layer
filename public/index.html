Image Sarch Abstraction Layer

Created according to the freeCodeCamp backend API Project Image Search Abstraction Layer.

Usage

  • See the results of a Google Image search by entering/api/imagesearch/<searchterm>. For example: /api/imagesearch/funny cats
  • Paginate through image search results by adding ?offset=<number>. Ten results starting from that number result will be displayed. For example, to get results 21 through 30 of the image search "funny cats": /api/imagesearch/funny cats?offset=21
  • See a list of the most recently submitted search strings by entering /api/latest/imagesearch
